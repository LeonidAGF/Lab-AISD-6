### **Вариант:**

####  №6 Кодирование по Цезарю всех текстовых файлов в указанном каталоге и его поддиректориях при помощи утилиты tr.  

### **_Что нужно было сделать_:**

Написать Bash скрипт, который получает путь до директории, в которой лежат файлы для кодирования, число символов, на которое будет производиться сдвиг по Цезарю. После получения всей необходимой информации скрипт должен закодировать все txt файлы по Цезарю  в указанном каталоге и его поддиректориях при помощи утилиты tr.

### **Как работает алгоритм**

Алгоритм запрашивает через консоль от пользователя путь до каталога, где нужно зашифровать файлы, и число, на которое будет производиться сдвига при  помощи строк с командами read .  Далее строит сдвинутые строки латиницы  через выделение подстрок. Затем с помощью find находятся все txt файлы и при помощи цикла while read -r file, в котором название файла помещается в переменную file, имеющим тело, в котором читается файл file, вызывается tr, которая заменяет символы в файле согласно сдвигу и сохраняет получившийся результат в переменную content. После при помощи echo в исходный файл записывается зашифрованный по Цезарю текст. Так цикл проходится по всем файлам, найденным при помощи find. После завершения цикла, программа выведет в консоль "done".
